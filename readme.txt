Just Start Package

== Description ==
- Admin Dashboard Theme Layouts.
- Admin Color Scheme (Dark Mode).
- Admin Widgets (Your System Information - Your Profile Information - Server Information).
- Maintenance Mode.
- Admin Language Switcher.
- Hello+ Lyrics Add-On.
- Advanced Custom Fields (Just Included, Thanks for ACF).
- Dashboard Options Page.
- Disable front-end theme.

== Installation ==
1. Upload \"oneds-start\" folder to the \"/wp-content/plugins/\" directory.
2. Activate the plugin through the \"Plugins\" menu in WordPress.
